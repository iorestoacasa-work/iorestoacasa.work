Aggiungi nuovi server

Acquista un server personalmente, oppure contatta università, provider ed istituzioni italiane e chiedi di mettere a disposizione un server.

Grazie a Docker puoi installare una nuova istanza di Jitsi Meet in pochi minuti, seguendo questa guida.

Se invece hai configurato Jitsi in autonomia, puoi esporre le metriche grazie a questo repository.

Se vuoi installare Multiparty-Meeting, puoi usare questo repository.


I requisiti minimi per partecipare:

• una macchina dedicata allo scopo, non condivisa con altri servizi.

• IPv4 statico (possibilmente in un datacenter italiano)

• connessione 100 Mbps simmetrica (meglio 1 Gbps, ogni utente occupa circa 4 Mbps)

• 4 CPU server grade (Xeon o analogo)

• 4GB di RAM


Esponi le metriche:

Per mostrare il numero di utenti connessi e migliorare la stabilità del servizio, ogni server nella tabella espone delle statistiche anonime.
