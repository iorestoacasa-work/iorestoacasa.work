Aggiungi nuovi server

Acquista un server personalmente, oppure contatta università, provider ed istituzioni italiane e chiedi di mettere a disposizione un server.

Grazie a Docker puoi installare una nuova istanza in pochi minuti, seguendo questa guida.

Se invece hai configurato Jitsi in autonomia, puoi esporre le metriche grazie a questo repository.


I requisiti minimi per partecipare:

• IPv4 statico (possibilmente in un datacenter italiano)

• connessione 100 Mbps simmetrica (meglio 1 Gbps, ogni utente occupa circa 4 Mbps)

• 4 CPU server grade (Xeon o analogo)

• 4GB di RAM


Esponi le metriche:

Per mostrare il numero di utenti connessi e migliorare la stabilità del servizio, ogni server nella tabella espone delle statistiche anonime.

Non abbiamo ancora messo a punto una procedura per installare Multiparty-Meeting.
Puoi seguire gli sviluppi (e dare una mano) in questo repository.

Integra la lista dei server nel tuo sito

Inserisci il seguente snippet nel tuo sito web per mostrare ai tuoi utenti la lista di server Jitsi che possono utilizzare. Se preferisci puoi attingere direttamente ai dati grezzi tramite il file /hosts.json

    <iframe src="https://iorestoacasa.work/table.html" height="176" width="300"
        style="border:0px;" name="iorestoacasa">
    </iframe>
</div>
